Git 101

status

ใช้ตรวจสอบสถานะของไฟล์

touch

touch [ชื่อไฟล์] คือ คำสั่งการสร้างไฟล์ใหม่

add.(Add Dot)

ใช้เพื่อ add File ทั้งหมดที่เรา Touch หรือสร้างมา
เวลาใช้ต้องด้วยความระมัดระวังเพราะบางทีไฟล์ที่เราไม่ต้องการ add อาจจะเข้าไปด้วย

git log ก็คือ git log
สำหรับดู History log

ignore

สร้างไฟล์ชื่อ .ignore

Edit ไฟล์ .igore ใส่ชื่อไฟล์ หรือ pattern ของไฟล์ที่ไม่ต้องการให้ git สนใจ

git init

คำสั่ง git init ใช้สำหรับเริ่มต้นสร้าง git repository

git branch

คำสั่ง git branch ใช้สำหรับแสดง branch ที่มีอยู่ใน repository ทั้งหมด

git merge

คำสั่ง git merge ใช้สำหรับรวม code ระหว่าง 2 branch

git push

คำสั่ง git push ใช้ upload code ขึ้น remote repo

Git Commiit

git commit -m "[descripttive message]"
สำหรับ commit หลังจาก add file

git add [file]"
เพิ่มไฟล์

mkdir

mkdir [directory name] คือ คำสั่งสร้าง Directory

reset

reset ใช้สำหรับยกเลิกไฟล์ที่ถูก add อยู่ใน stage

 pull

pull ใช้ดึง file ล่าสุดจาก storage

 log

Git Log ใช้แสดง history log
